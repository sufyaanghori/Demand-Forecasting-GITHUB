README

Each folder(Brazilian E-commerce, E-commerce Dataset, Sales of summer cothes) contains the original files provided by the author. 
Each folder has a notebook that performs EDA on that respective dataset in order to help compilation of the datasets. 
Final_dataset.csv consists of the final compiled dataset on which our model has been run. Notebook-Final Dataset contains the code 
of using the XGB regressor model on our final dataset.
